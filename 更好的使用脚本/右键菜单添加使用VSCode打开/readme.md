# 使用方法
1. 右键编辑"add_vscode_open.reg"，替换文件中的'D:\\\\software\\\\Microsoft VS Code'为本机VScode安装的路径
2. 双击"add_vscode_open.reg"文件即可
3. 如果有警告弹出，点击'是'即可
4. 添加成功后，右键文件/文件夹即可看到使用VSCode打开

# 所需运行环境
无

# 注意
1. 只需要替换路径即可，不需要替换'\\\\Code.exe'部分
2. 替换路径时注意，'\\'符号需要是两个来转义
