# 使用方法
1. 双击"右键菜单切换为Win10.bat"/"右键菜单切换为Win11.bat"文件即可
2. 如果有警告弹出，点击'是'即可
3. 切换成功后，会自动重启文件管理器

无

1. Windows 11 系统才需使用
2. 所有打开的文件夹会被关闭
3. 会清空剪切板中的内容
